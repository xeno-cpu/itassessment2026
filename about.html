About HMIS

HMIS is a modern, user-friendly web-based Health Management Information System developed specifically for hospitals and clinics in Nepal. It helps doctors, nurses, and administrators manage patient records, appointments, reports, and data analytics in real time.

Our Mission

To digitize healthcare in Nepal and make quality medical services faster, safer, and more accessible through technology.

Key Features

  • Secure electronic patient records
  • Real-time appointment booking
  • Advanced data analytics dashboard
  • Inventory and pharmacy management
  • Built with clean HTML, CSS & JavaScript

This website was created as the final project for the Web Design & Development course at The British College, Kathmandu.
