# 🏥 HMIS Branding Project

This project is a simple **HTML and CSS-based branding prototype** for a Hospital Management Information System (HMIS).  
It focuses on layout design, visual identity, and basic front-end structure.

---

## 🎯 Objective

The goal of this project is to:
- Create a clean and professional branding layout for HMIS
- Practice HTML and CSS fundamentals
- Demonstrate basic web design skills

---

## 🚀 Features

- Responsive page layout (basic)
- Custom styling using CSS
- Clean and minimal design
- Branding-focused UI (logo, layout, typography)

---

## 🛠️ Tech Stack

- HTML5
- CSS3
